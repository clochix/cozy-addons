Extensions Cozy

Cette application est juste un dépôt hebergeant quelques extensions qui ajoutent de nouvelles fonctionnalités aux applications de Cozy. Pour profiter de ces extensions, il est pour l'instant nécessaire d'installer une extension dans votre navigateur (l'extension n'est actuellement disponible que pour Firefox) et l'application Acthésis dans votre Cozy. Cette dernière permet à deux applications Cozy de communiquer.

Les fonctionnalités offertes sont les suivantes :

  • Emails
    • enregistrer dans votre calendrier les invitations à des évènements reçues ;
    • utiliser PGP pour signer et chiffrer vos messages ;
    • enregistrer dans l'application Files les pièces jointes d'un message ;
    • attacher à un nouveau message un fichier depuis Files ;

Le code source de ces extensions est disponible sur Github.
